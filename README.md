Insurance Claim Prediction

This project predicts the likelihood that a person will file an insurance claim using machine learning. The system takes user details (age, gender, BMI, number of dependents, smoking status, region, etc.), preprocesses the data, and uses a Random Forest classification model to predict claim probability.

Features:
Predicts whether an insurance claim is likely using a classification model with approximately 95% precision.
User-friendly command-line interface for quick input and instant prediction results.
Automatically handles data preprocessing, encoding, and scaling.
Provides probability scores alongside binary predictions for better decision-making.
Includes model evaluation metrics such as precision, recall, and accuracy for performance assessment.


Technologies: Python, pandas, scikit-learn, numpy
